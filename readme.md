## Steps to Push Code

1. Initialize Git
   git init

2. Add files
   git add .

3. Commit changes
   git commit -m "First Commit"

4. Set main branch
   git branch -M main

5. Add remote repository
   git remote add origin https://github.com/AshuDevWorks/React-JS.git

6. Push to GitHub
   git push -u origin main
